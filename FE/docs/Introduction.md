### Introduction

liteApp是由爱奇艺内部开发的跨平台容器框架，结合了web与native的优势，从多方面出发解决webview性能问题，赋予webview媲美native体验的能力


#### why liteApp
极致体验 ( native ) + 开发效率 ( h5 ) + 快速迭代 ( 动态化 ) = lite-app
